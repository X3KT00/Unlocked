# Unlocked
Это мессенджер который работает через Radmin VPN, я не думаю что его хоть как-то можно заблокировать.

# Для его работы нужно установить:
Radmin VPN,
Python,
и прописать pip install -r requirements.txt открыв коммандную строку в папке где находится файл requirements.txt

Вот файл README.txt из папки

Unlocked - мессенджер который не заблокируют (пока интернет не отключат)

Для пользования нужно скачать Radmin VPN и Python

Как начать пользоваться:
	1.Открой users.json
	2.Там есть пример пользователя, вам нужно создать на основе этого примера своих пользователей
	(себе и друга, я для редактирования json и py файлов использую Visual Studio Code)
	3.ну тип запускай unlocked.py хз, видео и картинки будут сохранятся в папки
	(видео - videos, картики - images)

если что сообщения будут сохранятся в messages.json
